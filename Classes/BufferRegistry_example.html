/*

 BufferRegistry example

 

 Creating

 

  BufferRegistry();

  BufferRegistry(\name);  

  BufferRegistry(\name, server);

 

  Names are unique, so calling BufferRegistry(\blah) either makes a new one or 

  returns the existing BR if it exists. BufferRegistry() defaults to \default, 

  which is generally the best to use.

 

  BufferRegistry creates buffers via read or alloc commands, and gives them

  a unique name. You can repeatedly assign and reassign buffers to the same

  name, and they're intelligently swapped out without causing a bunch of 

  duplicates. The buffers are respawned intelligently when a server is booted or

  rebooted. Buffers with one name will keep their bufnum for the duration of

  the server being alive.

 

 Using

 

  ~br = BufferRegistry();

  ~br.read(\b1, "/Users/scott/Desktop/S&M/_audio/yamaha mr-10/MR10ltom.wav");

  ~br.read(\b2, "/Users/scott/Desktop/S&M/_audio/yamaha mr-10/MR10ltom.wav");

  ~br.read(\b3, "/Users/scott/Desktop/S&M/_audio/yamaha mr-10/MR10ltom.wav");

 

  ~br[\b3].play;

  s.quit;

  s.boot;

  ~br[\b3].play;

 

*/
